## 什么是heap 
堆是一种特殊的树结构，它的根节点总是全局最小或者最大，自然也就衍生出了最大根堆和最小根堆

## go的heap实现
go的heap实现了如下接口
```go
type Interface interface {
	sort.Interface
	Push(x interface{}) // add x as element Len()
	Pop() interface{}   // remove and return element Len() - 1.
}
```
初始化时进行了如下操作
```go
func Init(h Interface) {
	// heapify
	n := h.Len() 
	for i := n/2 - 1; i >= 0; i-- {
		down(h, i, n)
	}
}
```
